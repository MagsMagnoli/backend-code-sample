# Express Server Starter

A Node.js express server starter project built with typescript

## Quick Start

Install the dependencies:

```
yarn install
```

Start the server:

```
yarn start
```

Access the tasks endpoint by visiting `localhost:3000/api/tasks` in your browser

## Integrations

- express
- typescript
- knex
- jest
- prettier
- eslint
- docker
- nvm
- path aliases

## Tests

Run the tests:

```
yarn test
```
